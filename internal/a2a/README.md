# A2A PoC — Federated Agent Card

This package is a proof-of-concept for adding [Agent-to-Agent (A2A) protocol](https://github.com/google-a2a/A2A) support to the MCP Gateway.

## What it demonstrates

The MCP Gateway federates tools from multiple upstream MCP servers into a single `/mcp` endpoint. A2A requires the equivalent for agents: a gateway that fetches Agent Cards from multiple upstream A2A agents and serves a unified card at `/.well-known/agent.json`.

`A2ABroker.FederatedCard()` is the core of this PoC. It:

1. Fetches `AgentCard` JSON concurrently from each registered upstream agent's `CardURL`
2. Applies the per-agent `SkillPrefix` to every skill ID (e.g. `forecast` → `weather_forecast`)
3. Merges all prefixed skills into a single card that advertises the gateway as the endpoint
4. Tolerates upstream failures — a failed card fetch is logged and skipped; healthy agents still contribute their skills

This mirrors exactly how the MCP broker federates tools via `OnConfigChange` → `upstream.MCPManager`.

## What is NOT included in this PoC

- **Router integration**: wiring `A2ABroker` into the ext_proc path in `internal/mcp-router/server.go` to detect and route `/a2a` traffic
- **CRD controller**: a `A2AAgentRegistration` reconciler in `internal/controller/` that resolves HTTPRoutes and calls `SetAgents()`
- **Task proxying**: forwarding `message/send` and `tasks/get` JSON-RPC calls to the correct upstream agent
- **Session handling**: A2A task IDs mapped to upstream task IDs (analogous to `idmap.Map` for elicitation)

## Running the tests

```bash
go test ./internal/a2a/...
```

## Files

| File | Purpose |
|------|---------|
| `types.go` | A2A protocol types: `AgentCard`, `Skill`, `AgentCapabilities`, `AgentAuth` |
| `broker.go` | `A2ABroker` interface and `FederatedCard()` implementation |
| `broker_test.go` | Unit tests using `httptest.Server` mock upstream agents |
| `../config/a2a_types.go` | `A2AAgent` config struct (mirrors `config.MCPServer`) |
| `../../api/v1alpha1/a2aagentregistration_types.go` | `A2AAgentRegistration` CRD type |
| `../../api/v1alpha1/a2aagentregistration_deepcopy.go` | DeepCopy methods (generated by `make generate-all` in a real PR) |

// cmd/mcp-broker-router/main.go
a2aBroker := a2a.NewA2ABroker(cfg.A2AAgents, externalHostname, logger, nil)
